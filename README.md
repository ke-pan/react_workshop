# React Workshop in Baozou

## 为什么，初心和私心
- 教是最好的学
- 督促自己
- 练习表达
- 找到伙伴

## 定位
- 入门教程
- 共同学习

## 有什么内容
- 从零开始写一些组件，写一个应用
- 从前端到后端
- 从桌面端到手机端

## 没有什么内容
- 不介绍 React 是什么
- 不讨论性能
- 不对比框架

## 需要什么准备
- 基本的 HTML，CSS 和 JS
- 接受讲者是一个菜鸟
- 一些时间
- 一台装有 node 的电脑
